## 📝 插件功能
1. 规格统计：可以统计商品评论区显示的所有评论内容，统计不同规格的评论数、占已统计总评论占比、差评数、差评率。
2. 差评内容：显示已加载的差评具体内容（3星及以下为差评），可导出到本地Excel。
3. 评论图片：显示已加载的用户评论图片，可批量下载图片到本地文件夹。
4. 好评内容：显示已加载的好评具体内容（4星及以上为好评），可导出到本地Excel。
